🧬 Суть проекта
L-NEURON — это математическая модель «умного нейрона», предназначенная для сведения неограниченного потока данных в единый Индикатор Системного Благополучия (0.0 — Норма, 1.0 — Катастрофа).
В отличие от стандартных нейросетей, L-NEURON не требует тысяч итераций обучения. Он базируется на экспертной патофизиологической логике, где критическое отклонение одного параметра важнее стабильности десяти остальных.
🛠 Ключевые алгоритмические принципы
1. Асимметричная чувствительность (The "Critical Drop" Effect)
Система запрограммирована «пугаться» падения показателей сильнее, чем их роста.
Дефицит ресурса (например, падение давления или нехватка топлива) усиливает вес отклонения в 4 раза по сравнению с избытком.
2. Принцип Доминанты (90/10)
При объединении группы параметров (напр., Гемодинамика, Датчики двигателя), итоговый индекс группы на 90% определяется самым худшим показателем (Max-Pooling) и лишь на 10% — среднеквадратичным фоном.
Средняя температура по больнице не информативна, если у одного пациента лихорадка 42°.
3. Иерархическая деградация (80/20)
Финальная сборка всех систем в единый индекс «светофора» отдает 80% приоритета самой пораженной системе («лидеру патологии»). Это позволяет мгновенно увидеть общую деградацию системы, даже если сбоит только один критический узел.
4. Функция Softsign-компрессии
Для стабилизации выходного сигнала используется нелинейная функция, которая мягко «прижимает» экстремальные значения к пределу 0.99, предотвращая математический взрыв системы.
📈 Сферы применения
Хотя алгоритм отточен на гемодинамике и интенсивной терапии, он универсален для любых Hard-Real-Time систем:
Медицина: Оценка полиорганной недостаточности (ЛЖ, ПЖ, почки).
Техногенный мониторинг: Контроль сложных инженерных узлов.
Экология: Интегральный индекс загрязнения среды.
Финансы: Детектор рыночного обвала.

## 🇷🇺 Описание проекта (Russian)
**L-NEURO** — это специализированный модуль на C++, предназначенный для оценки состояния сложных систем на основе многомерных данных. В отличие от стандартных методов, модуль имитирует логику живых систем (гомеостаз) и использует иерархическую агрегацию рисков.

### Ключевые особенности:
*   **Асимметричная нормализация:** Система в 4 раза чувствительнее к критическому падению параметров (`Critical Drop`), чем к их превышению.
*   **Зона гомеостаза:** Внутри границ нормы поддерживается микро-сигнал «живучести» (0.01), исключая «мертвые» нулевые зоны.
*   **Иерархия 90/10:** Локальная оценка группы параметров на 90% определяется худшим показателем (Max-Pooling) и на 10% — общим шумом (RMS).
*   **Системный индекс 80/20:** Финальный расчет тяжести состояния отдает приоритет ведущей патологии (80%), учитывая фон остальных систем (20%).

## 🇺🇸 Project Overview (English)
**L-NEURO** is a C++ module for evaluating the status of complex multidimensional systems. It mimics biological homeostasis and employs hierarchical risk aggregation.

### Core Features:
*   **Asymmetric Normalization:** The system is 4x more sensitive to parameter drops (`Critical Drop`) compared to overhead stress.
*   **Homeostasis Support:** Maintains a "live signal" (0.01) within normal ranges to ensure system dynamism.
*   **90/10 Hierarchy:** Group status is driven by the worst-performing metric (90%) and background noise (10%).
*   **80/20 System Index:** Final severity index focuses on the dominant pathology (80%) while considering the status of auxiliary systems (20%).
